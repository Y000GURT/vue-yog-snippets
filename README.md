## All prefixes
| Префикс  | Описание                                      |
|----------|-----------------------------------------------|
| `vbase`  | Создание базового Vue компонента             |
| `vpr` | Создание шаблона для пропсов                 |
| `vem` | Создание шаблона для эмитов                  |
| `vm`     | Создание шаблона для `onMounted`             |
| `vum`    | Создание шаблона для `onUnmounted`           |
| `vrt`    | Создание `route`                              |
| `vrtr`   | Создание `router`                             |
